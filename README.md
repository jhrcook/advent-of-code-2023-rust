## Setup

```bash
git clone git@github.com:jhrcook/advent-of-code-2023-rust.git
cd advent-of-code-2023-rust
```

## Execute puzzles

Dev build and run:

```bash
cargo run --release
cargo run --release -- --day 1
```

Full install and run:

```bash
cargo install .
advent-of-code-2023-rust
advent-of-code-2023-rust --day 1
```
